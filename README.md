# Zomato India Restaurant Intelligence

## Overview

This project analyzes Zomato restaurant data to uncover insights about pricing, ratings, and customer preferences.

## Key Features

* Data cleaning and preprocessing
* Feature engineering
* Exploratory Data Analysis
* Machine Learning model for rating prediction
* Business insights extraction

## Insights

* Online ordering has a positive impact on ratings
* Certain locations are highly saturated with low-performing restaurants
* Mid-range pricing yields optimal ratings

## Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib, Seaborn

## How to Run

```bash
pip install -r requirements.txt
python main.py
```
